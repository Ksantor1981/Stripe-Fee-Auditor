# Stripe Fee Auditor

MVP-сервис для анализа комиссий Stripe по CSV-экспорту Balance, выявления аномалий и подготовки отчета с freemium/paywall-моделью.

## Цель

Помочь бизнесу понять реальную комиссию Stripe, увидеть драйверы роста fee-rate и получить практические рекомендации по снижению издержек.

## Текущий статус

- Инициализирован каркас проекта.
- Подготовлены базовые директории и конфиги.
- Добавлен стартовый техспек в `docs/TECH_SPEC.md`.

## Базовая структура

- `config/` — конфигурация проекта
- `src/` — исходный код приложения
- `tests/` — тесты
- `scripts/` — служебные скрипты
- `data/`, `logs/`, `storage/`, `results/`, `temp/` — данные/артефакты/временные файлы
- `docs/` — документация и спецификации

## Быстрый старт

1. Установить зависимости:
   - `pip install -r requirements.txt` (для Python-утилит)
2. Инициализировать JS-приложение (Next.js) в рамках этого проекта:
   - `npx create-next-app@latest . --typescript --tailwind --app --src-dir=false`
3. Настроить переменные окружения:
   - `DATABASE_URL`
   - `BLOB_READ_WRITE_TOKEN`
